# 瓦罗兰蛇

这是一个基于控制台的贪吃蛇小游戏，支持角色解锁、积分商店与简单技能系统。

**源码**: [VALOSnake.cpp](VALOSnake.cpp)

**功能概览**
- 基础贪吃蛇玩法：移动、吃食物、变长、避免碰撞。
- 角色系统：普通贪吃蛇（白板）、炼狱（清屏技能）、不死鸟（免死技能）。
- 商店：使用积分解锁角色（默认炼狱与不死鸟被锁定，需要 500 积分购买）。
- 得分转积分：每局结束时，当前得分按 1:1 转化为积分，可在商店消费。

**开发环境 / 依赖**
- Windows（控制台程序，依赖 <windows.h>）
- 推荐使用 MinGW-w64 的 g++ 或者其他兼容的 Windows 下 C++ 编译器

**构建与运行**
在项目根目录（包含 `1.cpp`）打开命令行后执行：

```bash
# 使用 g++ 编译
g++ -g VALOSnake.cpp -o VALOSnake.exe

# 运行
./VALOSnake.exe
```

如果使用 VS Code 的任务（workspace 已配置），也可以直接使用任务 `C/C++: g++.exe 生成活动文件` 来编译并运行。

**操作与控制**
- 移动: `W`/`A`/`S`/`D` 或 方向键
- 技能: 按 `K`（仅在角色具有技能且能量足够时可用）
- 退出: `ESC`

**角色说明**
- 普通（默认）: 无特殊技能。
- 炼狱: 吃到食物时获得能量（+1）。按 `K` 消耗 10 点能量，清除场上所有食物（每个食物 +10 分并增加蛇长）。
- 不死鸟: 吃到食物时获得能量（+1）。按 `K` 消耗 10 点能量，获得 3 次免死机会（碰撞时自动使用，消耗一次免死并允许改变方向逃生）。

**商店与积分**
- 每局结束后，系统会把本局得分自动转为积分（`积分 += 得分`）。
- 商店中两位角色的解锁价格均为 `500` 积分。
- 商店入口在：
  - 游戏结束后选择进入商店
  - 或在角色选择界面选择“商店”进入

**注意事项与扩展建议**
- 当前实现未持久化积分或解锁状态（程序退出后会丢失）。如需保存，请扩展为将积分和解锁状态写入本地文件（例如 `save.dat` / JSON）。
- 可考虑增加配置文件以调整角色价格、技能消耗与行为。
